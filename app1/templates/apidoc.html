API Documentaion

List countries

GET /countries - list all countries in alphabetical order

GET /countries/<optional:countries_id> get specific country

Optional parameters

name (string) country name used to specify a single country

Countries Object
GET /country
returns

_id unique id generated from mongo data base


Example: "_id": {"$oid": "5ee3f7ceaf8743d0a16d6adf"}


name name of the country


Example: "name": "Afghanistan"


data data corresponding to the country, includes 3 csv's with year and data as key and value


Example: "data": {"child_mortality_0_5_year_olds_dying_per_1000_born": {"1800": "469"}


Example: "data": { "children_per_woman_total_fertility": {"1800": "7"}


Example: "data": {"income_per_person_gdppercapita_ppp_inflation_adjusted": {"1800": "603"}

Create New Country

POST /countries - creates a new country name

adding a string to the input creates a country with that name

currently no implementation to add data with country name

returns country to list if call succeded

Delete a Country

DELETE /countries

adding a string to the input will Delete a country with that name
